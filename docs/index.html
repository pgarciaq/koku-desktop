Cost Management Desktop

A lightweight native desktop client for Red Hat Cost Management. Connect to console.redhat.com or your private on-prem instance.

Download Source Code

Features

Everything you need, on your desktop

Built with Tauri v2 for a fast, lightweight experience. No Electron bloat — uses your system’s native webview.

Connection Profiles

Save multiple named server configurations — SaaS and private on-prem instances — and switch between them instantly.

Flexible Authentication

Offline Token, Username & Password, Service Account, or Developer mode. Works with Red Hat SSO and any Keycloak instance.

Light & Dark Themes

Choose light, dark, or follow your system preference. The unified titlebar adapts seamlessly to both themes.

Unified Titlebar

Menus, window controls, and user profile merged into a single compact bar — inspired by JetBrains IDEs.

System Tray

Minimize to the system tray. Quick access to restore the window, open settings, or quit from the tray menu.

Native File Dialogs

CSV exports open a native “Save As” dialog. Print any page with Ctrl+P using your system printers.

Connection Profiles

One app, many instances

Save named profiles for Red Hat Hybrid Cloud Console (SaaS) and any number of private on-prem Cost Management instances. Each profile stores its own server URL, authentication method, and credentials. Switch with a single click.

The built-in SaaS profile is always available and cannot be deleted. Create as many private profiles as you need.

Authentication

Four ways to connect

Choose the auth method that fits your environment:

  • Offline Token — personal Red Hat account, recommended for SaaS
  • Username & Password — Keycloak password grant, best for on-prem
  • Service Account — client credentials grant, for automation
  • Developer mode — no auth, for local development

Get Started

Installation

Build from source or install a package.

Build from source

git clone https://github.com/pgarciaq/koku-desktop.git
cd koku-desktop
./scripts/build-ui.sh
npx tauri build

Install RPM (Fedora / RHEL)

sudo rpm -i "src-tauri/target/release/bundle/rpm/\
Cost Management Desktop-0.1.0-1.x86_64.rpm"

Install DEB (Debian / Ubuntu)

sudo dpkg -i "src-tauri/target/release/bundle/deb/\
Cost Management Desktop_0.1.0_amd64.deb"
